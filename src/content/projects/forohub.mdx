## Overview

**ForoHub** is a comprehensive REST API for educational forum management, developed as part of the Oracle ONE technical challenge. The system enables seamless interaction between students and instructors through course-based discussions, topic management, and collaborative learning features.

**Repository**: [github.com/luiscm17/ForoHub-APIRest](https://github.com/luiscm17/ForoHub-APIRest)

---

## Challenge Context

**Oracle ONE Backend Challenge** required building a complete REST API following industry best practices and modern Java development standards. The project demonstrated proficiency in Spring Boot, database design, and RESTful architecture while solving real-world educational platform requirements.

**Technical Requirements**: Build a scalable, maintainable API with proper separation of concerns, database migrations, and comprehensive documentation.
